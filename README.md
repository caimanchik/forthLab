## Задание 3. Визуальная модель работы перцептрона

Была создана еще одна сцена в Unity. В ней созданы плоскость, два куба, а также два материала - для 0 и 1. Создан скрипт [`ChangeColor`](https://github.com/caimanchik/forthLab/blob/main/ChangeColor.cs), который был добавлен плоскости. В нем был реализован метод `DotProductBias`, который использует веса для `OR` перцептрона. Когда оба кубика падают на плоскость, скрипт смотрит на цвета кубиков и принимает решение, в какой цвет окрасить плоскость в соответствии с `OR` логикой
